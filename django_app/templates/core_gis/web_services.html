Web Services

Os serviços web seguem as especificações da Infraestrutura Nacional de Dados Espaciais (INDE) e padrões OGC (Open Geospatial Consortium).
WMS - Web Map Service

Serviço para visualização de mapas em formato de imagem (tiles). Ideal para visualização em aplicações web e software GIS.

URL do Serviço:
{{ geoserver_url }}/pvh/wms
Camadas Disponíveis:
  • pvh:desmatamento_pvh - Desmatamento PRODES
  • pvh:alertas_deter - Alertas DETER
  • pvh:focos_calor - Focos de Calor
  • pvh:bairros_pvh - Bairros de Porto Velho

WFS - Web Feature Service

Serviço para acesso a dados vetoriais (features). Permite consultas espaciais e download de dados.

URL do Serviço:
{{ geoserver_url }}/pvh/wfs
Formatos de Saída:
  • GeoJSON
  • GML
  • Shapefile
  • KML

API REST

API RESTful com suporte a GeoJSON para desenvolvedores. Documentação interativa disponível via Django REST Framework.

Base URL:
/api/
Endpoints:
  • /api/desmatamento/
  • /api/alertas/
  • /api/focos/
  • /api/bairros/
  • /api/municipios/
Documentação API
Exemplos de Uso
Leaflet (JavaScript):
L.tileLayer.wms("{{ geoserver_url }}/pvh/wms", {
    layers: 'pvh:desmatamento_pvh',
    format: 'image/png',
    transparent: true
}).addTo(map);
QGIS:

Layer → Add Layer → Add WMS/WMTS Layer
URL: {{ geoserver_url }}/pvh/wms

Python (requests):
import requests
r = requests.get('/api/desmatamento/')
data = r.json()
