Sobre o GeoPortal

O que é o GeoPortal Porto Velho?

O GeoPortal Porto Velho é uma plataforma de dados geográficos desenvolvida para disponibilizar informações espaciais sobre o monitoramento ambiental e territorial do município de Porto Velho, capital do estado de Rondônia.

A plataforma integra dados de diversas fontes, incluindo o PRODES (Projeto de Monitoramento do Desmatamento na Amazônia Legal por Satélite) e o DETER (Sistema de Detecção de Desmatamento em Tempo Real), ambos desenvolvidos pelo INPE.

Funcionalidades
  • Mapas Interativos: Visualização de dados geográficos com Leaflet
  • Dashboards: Gráficos e estatísticas com Chart.js
  • Web Services: Acesso via WMS, WFS para integração GIS
  • API REST: Dados em formato GeoJSON para desenvolvedores
  • Downloads: Arquivos em formato shapefile e GeoJSON
Tecnologias Utilizadas
  • Django + GeoDjango
  • PostgreSQL + PostGIS
  • GeoServer (WMS/WFS)
  • Leaflet.js
  • Chart.js
  • Redis + Nginx
Fontes de Dados
  • TerraBrasilis - INPE
  • PRODES - Programa de Cálculo do Desflorestamento da Amazônia
  • DETER - Sistema de Detecção do Desmatamento em Tempo Real
  • Programa Queimadas - INPE
